Home | Workflows | Content | Channels | Analytics | Calendar

"Home" is the default active tab — shows the current dashboard view (Needs Attention, What's Working, Quick Metrics, New Leads). This is exactly what's already there, just relabeled under the "Home" tab.

"Workflows" tab — shows a grid of workflow cards. Each card has an icon, title, and short description with a "Start →" button. The workflow cards are:
  - 📝 Create Content Batch — "Generate a full week of content"
  - ⚡ Quick Post — "One post, any topic"  
  - 🚀 Launch Campaign — "Full launch: content + email + ads + funnels"
  - 🎥 Build Webinar — "Script + slides + follow-up sequence"
  - 📊 Monthly Strategy Refresh — "Performance review + next month plan"
  - ✉️ Build Email Sequence — "Welcome, nurture, promo, or any type"
  - 📢 Create Ad Set — "Hooks + creatives + copy variants"
  - 🧲 Build Lead Magnet — "PDF guide, quiz, or assessment"

"Content" tab — shows a thumbnail grid of content assets with filter pills across the top (All, IG, FB, Email, Ads, Blog) and status badges on each thumbnail (Published ✅, Scheduled 🕐, Draft 🔄, Pending Review 📋).

"Channels" tab — shows connected platform cards: Instagram (connected ✅, showing follower count and engagement rate), Facebook (connected ✅), Email Platform (connected ✅, showing list size), Meta Ads (connected ✅, showing active campaigns). Each card has a settings gear icon.

"Analytics" tab — shows the performance metrics view with a time range toggle (7 Days | 30 Days | 90 Days), a small engagement trend line chart, and metric groups for Content Performance, Email Performance, Ad Performance, and Top Performing content list.

"Calendar" tab — shows a weekly content calendar grid view with content items placed on specific days, color-coded by type (teal for posts, purple for email, orange for ads).

Show the "Home" tab as active/selected with the current dashboard content visible.
